\##  V5 — 一條龍穩定版



\*\*架構：\*\*

\- 單一 PHP WebSocket 伺服器，整合 AI、TTS、搜尋等功能。 

\- 一條線處理所有互動： → AI 回覆 → 語音 播放。



\*\*使用場景：\*\*

\- 多人小型聊天室（約 10～ 50人）。  

\- 快速部署教學／對話／朗讀／語音示範系統。  



\*\*優點：\*\*

\- 安裝簡單，一鍵啟動。  

\- 所有功能均內建（AI、TTS、搜尋）。  

\- IDE 友好、除錯迅速。



\*\*缺點：\*\*

\- 高併發下效能受速率限制。  

\- 外部 API 請求頻繁時成本偏高。
